## About The Project

NP NEPSE Live Market Dashboard is a web-based stock analysis tool built using Python and Streamlit.

This dashboard fetches live NEPSE market data and provides:

- Market summary
- Stock search functionality
- Full market overview table
- Highest & lowest price tracking

- Streamlit

- Streamlit Community Cloud

- 📈 Live NEPSE Data
